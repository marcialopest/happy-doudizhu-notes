# happy-doudizhu-notes

这是一个用于归档和发布多个独立 HTML 页面的仓库。  
所有页面均为独立文件，不依赖特定域名或后端服务，适合作为笔记、工具或演示页面的存放地。

## 项目简介

本仓库主要用于保存一些零散的 HTML 页面，内容涵盖日常记录、学习笔记、小工具原型等。  
每个页面均为自包含的 HTML 文件，可直接在浏览器中打开使用。

## 目录说明

```
/
├── README.md          # 本文件
├── index.html         # 仓库索引页（可选）
└── pages/             # 归档页面存放目录
    ├── example1.html
    ├── example2.html
    └── ...
```

- `index.html`：可选的主页，用于展示仓库内可用页面的简单列表。
- `pages/`：存放所有独立 HTML 页面的目录。每个文件应保持自包含，不依赖外部资源（除非通过 CDN 引用）。

## 页面归档说明

- 每个 HTML 页面应尽量保持独立，CSS 和 JavaScript 建议内联或通过公共 CDN 加载。
- 页面标题应清晰反映内容，便于索引和查找。
- 如有图片或其他资源，建议使用 base64 编码或托管在稳定可靠的图床（不在此仓库中直接上传大文件）。

## 维护说明

- 本仓库接受新增页面，但请确保页面内容不违反 GitHub 社区准则。
- 不建议在此仓库中存放需要后端支持或包含敏感信息的页面。
- 欢迎通过 Pull Request 提交新的页面，或对已有页面进行改进。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/happy-doudizhu-notes.git
   ```
2. 直接在浏览器中打开 `pages/` 下的任意 HTML 文件即可使用。
3. 也可通过 GitHub Pages 部署，访问仓库的 Pages 地址查看在线版本。
